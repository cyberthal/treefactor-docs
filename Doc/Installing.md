% How to Install

Treefactor's code is on [Github](https://github.com/cyberthal/treefactor).

MELPA
------

If you installed from MELPA, you're done. (This package is not yet on MELPA.)

Manual
-------

Put this file in your load-path:
trs.el

Put this in your init file: 

~~~
(require 'treefactor)
~~~
